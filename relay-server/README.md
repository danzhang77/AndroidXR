# Android XR Relay Server

Run from this directory:

```sh
npm install
npm start
```

The relay listens on `ws://0.0.0.0:8787`.

Open the browser demo at:

```text
http://localhost:8787
```

Use `ws://localhost:8787` from the browser on the laptop. Use `ws://<laptop-lan-ip>:8787` from the Android phone.
